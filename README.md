##  What is the NCA 6 Certification?

The **Nutanix Certified Associate (NCA) 6** is an industry-recognized certification offered by **Nutanix** — one of the world's leading companies in **Hyper-Converged Infrastructure (HCI)** and **multicloud solutions**.

This certification validates that the holder has:
- A solid understanding of the **Nutanix AOS (Acropolis Operating System) 6.xx** platform
- The ability to **navigate, manage, and operate** Nutanix clusters in real-world environments
- Hands-on skills across **compute, storage, networking, and virtualization** within a Nutanix environment
- Knowledge of **multicloud strategies** and how Nutanix fits into modern enterprise IT architectures



---

##  Why Nutanix? — Company & Technology Overview

**Nutanix** is a global leader in cloud infrastructure, headquartered in San Jose, California. It pioneered the **Hyper-Converged Infrastructure (HCI)** model, which combines compute, storage, and networking into a single, software-defined platform — eliminating the need for traditional siloed hardware.

### Core Nutanix Products relevant to NCA 6:
| Product | Purpose |
|---|---|
| **Nutanix AOS** | Core operating system for the Nutanix cluster |
| **Nutanix Prism** | Unified management console (Prism Element & Prism Central) |
| **AHV (Acropolis Hypervisor)** | Nutanix's built-in, free hypervisor |
| **Nutanix Files** | File storage services on HCI |
| **Nutanix Volumes** | Block storage services |
| **Flow Networking** | Software-defined networking & microsegmentation |

##  Skills Validated 

### 1.  Cloud Computing
Understanding of core cloud concepts including **IaaS, PaaS, SaaS**, deployment models (public, private, hybrid, multicloud), and how Nutanix enables cloud-like simplicity on-premises and across clouds.

### 2.  Cloud Computing & Virtualization
Expertise in **virtualization fundamentals** — how VMs are created, managed, and optimized — and how cloud-native workloads run on a virtualized Nutanix platform using **AHV or third-party hypervisors** like ESXi and Hyper-V.

### 3.  Computer Systems Administration
System-level knowledge including **cluster configuration, node management, software upgrades (LCM - Life Cycle Manager)**, hardware monitoring, and day-to-day administrative tasks within a Nutanix environment.

### 4.  Configure Virtual Networks
Ability to create and manage **virtual networks, VLANs, bridges, and virtual switches** within the Nutanix cluster using **Prism Element / AHV networking** to provide connectivity to workloads.

### 5.  HCI — Hyper-Converged Infrastructure
Core understanding of the **HCI architecture** where compute (CPU/RAM), storage, and networking are combined into a single appliance/node. Knowledge of how Nutanix **DSF (Distributed Storage Fabric)** distributes data across nodes for performance and resilience.

### 6.  Hyper-Converged Infrastructure Operations
Operational tasks specific to HCI — **adding/removing nodes, expanding clusters, monitoring cluster health**, understanding replication factors (RF2/RF3), and handling node failures without downtime.

### 7.  Managing Virtual Machines (VMs)
End-to-end VM lifecycle management — **creating, cloning, snapshotting, migrating, and deleting VMs** using Nutanix Prism. Includes configuring VM resources (vCPU, RAM, disk, NICs) and managing VM templates.

### 8.  Multi Cloud Strategy
Understanding how Nutanix **Xi Cloud / NC2 (Nutanix Cloud Clusters)** extends on-prem workloads to public clouds like **AWS and Azure**. Knowledge of hybrid and multicloud governance, workload portability, and unified management via Prism Central.

### 9.  Nutanix Platform Fundamentals
Deep knowledge of Nutanix's core architecture — **CVM (Controller VM)**, Stargate (storage I/O), Cassandra (metadata), Curator (MapReduce), Prism (UI/API), and how these components work together to deliver a resilient platform.

### 10.  Nutanix Clusters
Understanding of **cluster creation, expansion, health monitoring, and management** including node types (NX, Dell XC, HPE ProLiant DX, etc.), cluster networking requirements, and Nutanix's distributed architecture.

### 11.  Storage Administration
Managing storage within a Nutanix cluster — **Storage Pools, Containers (datastores), storage policies**, deduplication, compression, erasure coding (EC-X), and tiering strategies (SSD + HDD hybrid or All-Flash).

### 12.  Storage Virtualization
Knowledge of how Nutanix **abstracts physical storage** using DSF (Distributed Storage Fabric) to present a unified, high-performance storage layer to VMs — similar to how a SAN/NAS works but fully software-defined and distributed.

### 13.  Workload Migrations
Skills in migrating **existing workloads to Nutanix** using tools like **Nutanix Move (Xtract)** — supporting migrations from VMware vSphere, Hyper-V, AWS, and Azure to Nutanix AHV with minimal downtime.
